The Dynamic Portion

For this example the servlet is pulling data from a csv file and sending it back with both XML and JSON formats.

Both seem to be about the same when metrics are run, and neither is hard to work with on the client side, I guess I prefer XML just a little because t's a markup language so there's more consistency with the rest of the Javascript code running client side after we get the data. In general though, I set up applications to return xml, json, csv, and a custom/unique format for compatibility with other software.
